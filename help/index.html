1. HTML Structure - top

This theme has fluid layout with with maximum of 1300px width and padding of 20px on both sides. First is header, contains 3 parts, top part with some info and social icons, bottom part has logo (image or text) and primary menu on the right side. Page wrapper has 2 parts, main content and sidebar. Main content can be on left or right side and can be full width by setting the class of main content. The general template structure is the same throughout the entire template.

Folders inside template:

  • css - all css files for template
  • demo - all demo images, can be deleted
  • fonts - Fontawesome retina ready icons/fonts
  • js - all scripts for template like slider, lightbox, parallex effect etc.

2. Header Structure - top

Top part

In top part you can have some info like location and phone number on the left side. On the right side you can include social icons.

Bottom part (header)

Sticky header has 2 modes, with black bakcground with opacity, and white with opacity when you start scrolling. In white mode, logo will change to second logo, dark logo, so in yur folder you need to have 2 logos, one with white and one with dark colors. On the right side of logo there is search icon for open search form.

3. Layer Slider - top

With template comes premium Layer Slider. Documentation for slider is in separate folder slider-help.
